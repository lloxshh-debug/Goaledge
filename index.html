Setup Guide
Step 1 — Get Your Free API Key
1. Go to dashboard.api-football.com
2. Click Sign Up — just email + password
3. Copy your API Key from the dashboard
4. Free plan = 100 requests/day ✓
Step 2 — Deploy to Vercel (Free)
1. Go to vercel.com → Sign up with GitHub
2. Click Add New Project
3. Upload the goaledge folder you downloaded
4. In Environment Variables add:
  Name: FOOTBALL_API_KEY
  Value: your key from Step 1
5. Click Deploy — done! ✓
Step 3 — Connect the App
After deploying, Vercel gives you a URL like:
your-app.vercel.app
The app will automatically fetch live data from that URL. That's it!
